# _archive

Material kept in the repo for reference but **excluded from the marketplace**
(it lives outside `plugins/`, so the catalog scan and validators skip it).

## Contents

| Folder | What | Why archived |
|---|---|---|
| `pps-web-profile/` | The Aware `pps-web` profile — 4 generated agents (`web-implement`, `web-polish`, `web-pre-commit`, `web-test`) | A filled-in profile is project-specific. It is kept as a worked example of what `/profile-generator` produces, not as a published plugin. |

To use this kit on your own project, don't install this — run `/profile-generator`
(from the `react-agents` plugin) and generate a profile into your project's repo.
